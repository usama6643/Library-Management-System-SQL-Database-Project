# Library-Management-System-SQL-Database-Project
A SQL-based Library Management System demonstrating relational database design, joins, aggregate functions, and subqueries using PostgreSQL.

This project is a SQL-based Library Management System designed to manage branches, employees, members, and books. It demonstrates the use of relational database concepts including:

Table creation

Primary and Foreign Keys

Joins

Aggregate Functions

Group By & Having

Subqueries

Data Filtering

# Features Implemented

Display employees and their details

Find books by author

Count books by category

Find highest paid employee

Branch-wise salary expense

Employees earning more than branch average

Subqueries and Joins

Aggregate analysis

# SQL Concepts Covered

SELECT

WHERE

GROUP BY

HAVING

ORDER BY

JOIN (INNER & LEFT)

Subqueries

Aggregate Functions (COUNT, SUM, AVG, MAX)

CASE Statements
